## 🙏Ground Rule
### 기본적인 Rule

- ex) 일과 시간 12:00 ~ 22:00 (월 ~ 금) / 10:00 ~ 18:00 (토) - 오프라인 최소 주 4회

<br>

### 개발 일지 작성

> ex )Github Actions 활용해서 개발 시작하면서부터 작성

- 어제 할 일 & 오늘 한 일
- 구현 내용
- 공유할 내용 (Optional)


### Code Convention

- #### 파일명 - Upper Case
- #### 함수명 - UpperCamelCase
- #### 변수명 - LowerCamelCase
- #### 주석 - 주석은 최소 function or Method 단위로 작성한다
- #### 들여쓰기 - 들여쓰기는 space 4칸으로 설정한다
